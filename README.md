#Usage
```sh

offdroid init #intialize an offdroid "repo"
offdroid fetch-xmls #addons_list-2.xml, repositor-12.xml, etc.
offdroid list-packages #list the actual packages contained in the fetched xmls
offdroid create-barebones-sdk sdk/ from tools.zip
offdroid set-sdk sdk/
offdroid select-packages 
offdroid add-source xml-list.filemap as 'xml'
offdroid list-sources 
offdroid rm-source xml
offdroid launch-manager
```
